
# Parking Lot Detection Project

## Overview
This project focuses on detecting and counting cars in a parking lot using computer vision techniques.

## Dataset
The dataset used in this project was obtained from Kaggle using kagglehub.

## Files in this repository
- notebook.ipynb: Main Colab notebook containing the code
- report.pdf: Final project report
- requirements.txt: List of required Python libraries
- data/: Dataset folder used for training and testing

## How to run
1. Open the notebook in Google Colab or Jupyter Notebook
2. Install required libraries using requirements.txt
3. Run the notebook from top to bottom
